furniture-parsers
Скрипты на Python для парсинга каталогов мебели и дверей с возобновлением при внезапном падение скрипта и обходом блокировок.

Набор парсеров для мебельных сайтов (Mebel.com & Dveri.com)

Комплексное решение для сбора данных из интернет-магазинов. Скрипты оптимизированы для стабильной работы с большими объемами данных.

Основные возможности:
Обход защиты: Использование `curl_cffi` для имитации реального браузера
Система чекпоинтов: Автоматическое сохранение прогресса. Если интернет пропадет, парсер продолжит с того же места
Медиа-контент: Автоматическое скачивание изображений товаров в высоком разрешении
Структурированные данные: Экспорт в CSV с корректной кодировкой для Excel

Стек используемых библиотек
BeautifulSoup4, curl_cffi, lxml, tqdm.

Как запустить:
1. Установите библиотеки: `pip install -r requirements.txt`
2. Запустите нужный скрипт: `python parsers/mebel_parser.py`
